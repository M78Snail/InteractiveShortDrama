# 互动影视游戏项目

## 项目概述
基于AI驱动的互动影视游戏开发平台，结合 AI 生图、AI 视频生成技术，快速创作互动式剧情游戏。

## 技术栈
- **前端游戏引擎**: Flutter (Android/iOS 双端)
- **剧本工具**: 纯网页工具 (HTML+JS)
- **AI 素材**: ComfyUI (本地) + 即梦 (第三方)
- **参考作品**: 《隐形守护者》、《情感反诈模拟器》

## 目录结构
```
PersonalAi/
├── interactive_game/    # Flutter 游戏项目
├── story_tool/          # 网页剧本拆解工具
└── docs/                # 文档
```

## 工作流程
1. 写剧本 (Markdown 格式)
2. 用工具拆解为 JSON + 素材清单
3. AI 生成素材 (图片/视频)
4. 素材导入 Flutter 项目
5. 编译发布

## 开发规范
- 剧本使用 Markdown 格式
- 素材命名清晰规范
- 先跑通 Demo 再迭代
